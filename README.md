# 🛡️ NIST IR 8547 PQC Transition Knowledge Base

ยินดีต้อนรับสู่ Repository สรุปเนื้อหาเอกสาร **NIST IR 8547 ipd** ฉบับทำความเข้าใจง่าย โดยเนื้อหาทั้งหมดเรียบเรียงขึ้นจากเอกสารมาตรฐานต้นฉบับ ผสมผสานกับแนวทางการปฏิบัติจริง เพื่อให้เป็นแหล่งข้อมูลที่เข้าถึงง่ายสำหรับผู้ที่สนใจด้าน Post-Quantum Cryptography (PQC)

---

## 🎯 วัตถุประสงค์ของโปรเจกต์ (Project Objectives)

1. **เพื่อสรุปเนื้อหา**: รวบรวมสาระสำคัญจากเอกสาร NIST IR 8547 เรื่องการเปลี่ยนผ่านสู่มาตรฐานรหัสลับยุคหลังควอนตัมให้กระชับและตรงประเด็น 
2. **เพื่อการเตรียมความพร้อม**: ใช้เป็นเครื่องมือในการศึกษาวางแผนการย้ายระบบ (Migration) เพื่อรับมือกับภัยคุกคามจากคอมพิวเตอร์ควอนตัม 
3. **เพื่อวิเคราะห์ความเสี่ยง**: อธิบายผลกระทบและภัยคุกคามประเภท "Harvest Now, Decrypt Later" เพื่อให้เห็นภาพความจำเป็นในการเปลี่ยนผ่านที่เร่งด่วน 

---

## 📂 โครงสร้างภายในโปรเจกต์ (Project Structure)

เนื้อหาถูกแบ่งออกเป็นไฟล์ย่อยเพื่อให้ง่ายต่อการศึกษา ดังนี้:

* 📘 **01-PQC-Overview**: อธิบายภาพรวม ความสำคัญของ PQC และบทบาทของ NIST ในการกำหนดมาตรฐานใหม่ 
* 🔍 **02-Background**: อธิบายมาตรฐานรหัสลับและส่วนประกอบทางเทคโนโลยี
* ⏳ **03-Migration-Timeline**: อธิบายกรอบเวลาตามประกาศ NSM-10 และขั้นตอนการยกเลิกใช้งานอัลกอริทึมที่เสี่ยงภัย (Deprecated/Disallowed) 
* ⚙️ **04-Implementation-Guidance**: แนวทางการประยุกต์ใช้ในองค์ประกอบต่างๆ เช่น TLS, PKI และการใช้โซลูชันแบบผสม (Hybrid)
